# Relay

`aria relay` is the operator surface for paired-device control.

## Current Commands

- list devices
- register or revoke a device
- attach or detach a device from a runtime session
- queue a follow-up message
- queue an approval response
- inspect queued events
- mark delivery state

## What Relay Is For

- paired-device attachment to active runtime sessions
- remote follow-up messages
- remote approval responses

## What Relay Is Not For

- starting a second runtime
- owning tracked work
- bypassing runtime approval or policy logic
